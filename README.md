## 1.- [INSTALACION]

- Clonar el proyecto noticias
- $ cd path/to/clone/project/noticias
- $ composer install
- $ cp .env.example .env
- $ php artisan key:generate
- $ npm install
- $ npm run dev

## 2.- [NOTA]

Agregar esta constante al final del archivo .env con su respectivo valor

- NEWS_APP_KEY=your_news_api_key
