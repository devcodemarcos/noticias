## 1.- [INSTALACION]

- Clonar el proyecto noticias
- $ cd path/to/clone/project/noticias
- $ composer install
- $ cp .env.example .env
- $ php artisan key:generate
- $ npm install
- $ npm run dev

## 2.- [NOTA]

Agregar estas dos constantes con sus respectivos valores al final del archivo .env

- NEWS_APP_KEY=your_news_api_key
